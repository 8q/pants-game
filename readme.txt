■ゲーム概要
パンツを集めて世界を救え！
エージェントになってパンツを集めるゲームです。
たくさんのパンツを集めて3人の美少女達と仲を深めると、
彼女達のパンツが手に入るかも…？


■インストール
ファイルをダウンロードしてそのまま展開してください。
PickUpPants_Dataフォルダと、PickUpPants.exeが同じフォルダにない場合、起動することができませんのでご注意ください。

■アンインストール
展開したファイルをすべて削除するだけでアンインストールできます。


■操作方法

《アクションパート》

敵をよけつつパンツをたくさん集めて扉に入ればクリアです。
5回ダメージを受けるか穴に落ちるとゲームオーバーです。
使用するキーは4つです。

〇左へ移動
十字キー左 / A

〇右へ移動     
十字キー右 / D　

〇大ジャンプ
十字キー上 / W / X / スペース

〇小ジャンプ
十字キー下 / S / Z / SHIFT

「十字キー左右＋X＋Z」や「十字キー左右＋スペース＋SHIFT」、「WADS」といった組み合わせがおすすめです。
自分の操作しやすい方法を選んでください。


《ノベルパート》

ノベルパートのすべての操作はマウスクリックでも可能です。

〇次へ進む（長押しで早送りになります）
エンターキー / Z

〇スキップ
CTRL長押し

〇選択肢
マウスでクリック
