■操作方法

《アクションパート》

敵をよけつつパンツをたくさん集めて扉に入ればクリアです。
5回ダメージを受けるか穴に落ちるとゲームオーバーです。

使用するキーは4つです。

〇左へ移動
十字キー左 / A

〇右へ移動     
十字キー右 / D　

〇大ジャンプ
十字キー上 / W / スペースキー

〇小ジャンプ
十字キー下 / S / シフトキー

「十字キー左右＋スペース＋シフト」や「WAD＋シフト」といった組み合わせがおすすめです。
自分の操作しやすい方法を選んでください。
操作方法は設定から変更することもできます。
